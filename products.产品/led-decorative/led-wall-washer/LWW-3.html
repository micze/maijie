This product is mainly used for disco stage high lighting and decorating in building frame etc. It can work in independent mode, master/slave mode and DMX mode, which can generate various color changing effects, such as flashing, fading, steady ,seven color jumping synchronously etc. 256 classes gray degree dimmer，DMX programmable （thousands of effects available if many lights connected together）.
