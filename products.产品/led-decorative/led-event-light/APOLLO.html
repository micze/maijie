BATTERY WIRELESS DMX LED PAR-CAN
BATTERY LED EVENT LIGHTING
The APOLLO is a battery operated and wireless DMX controlled LED par-can. It mainly used for decoration and highlight in building, garden, stage, event, restaurant and architectural lighting, which can generate various color changing effects.

Size: 260mm × 190mm × 250mm

B. Features
● 3, 4, 5, 6, 7, 8, 9 channels 3/4/5/6 color RGB LED par-can
● Operating modes: Independent mode, Master/Slave mode, Standard DMX mode
● 3in1 / 4in1 / 5in1 / 6in1 LEDs
● 3-pin DMX input and output connectors
● Wireless DMX transmitter and receiver built in
● Seven groups independent Wireless Control
● Up to 12 hours battery life
● RGB color mixing with or without DMX control
● Color temperature presets (2,900~10,000 K)
● Remaining Power digital displayed
● Cooling fan built in
