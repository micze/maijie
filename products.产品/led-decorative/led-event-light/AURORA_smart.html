BATTERY WIRELESS DMX LED WALL WASHER
BATTERY LED EVENT LIGHTING
The AURORA smart is a battery operated and wireless DMX controlled LED wash light. It mainly used for decoration and highlight in building, garden, stage, event, restaurant and architectural lighting, which can generate various color changing effects.

B. Features
● 3, 4, 5, 6, 7, 8, 9 channels 3/4/5/6 color RGB LED wash light
● Operating modes: Independent mode, Master/Slave mode, Standard DMX mode
● 3in1 / 4in1 / 5in1 / 6in1 LEDs
● 3-pin DMX input and output connectors
● Wireless DMX transmitter and receiver built in
● Seven groups independent Wireless Control
● Up to 12 hours battery life
● RGB color mixing with or without DMX control

● Cooling fan built in
